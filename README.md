## Online Course Examination Learning Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/5ddcaea2d3104f59ba7aedf3bf5c133c/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users (students), teachers, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Discussion
5. Announcement Information
6. Courses
7. Exam Paper List
8. Personal Center
9. Exam Records
10. Wrong Answer Book
11. Background Management

**Teacher Features:**
1. Login
2. Change Password
3. Personal Information
4. Discussion Management
5. Announcement Information Management
6. Course Management
7. Course Video Management, Course Video Comment Management
8. Student Management
9. Assignment Management (Assignment Management, Answer Management)
10. Exam Paper Management
11. Question Management
12. Exam Management (Exam Records, Wrong Answer Book)



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Class Type Management, Discussion Type Management, Announcement Type Management, Course Type Management)
4. Discussion Management
5. Announcement Information Management
6. Course Management
7. Course Video Management, Course Video Comment Management
8. Teacher Management, Student Management
9. Assignment Management, Answer Management
10. Exam Paper Management
11. Question Management
12. Exam Management (Exam Records, Wrong Answer Book)
13. Carousel Information
